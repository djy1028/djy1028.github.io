发表于 2017-06-28 | 分类于 Vue.js | | 阅读次数

脚手架搭建

1.windows上安装git、node

2.利用命令行安装Vue,这里注意可以借助淘宝的npm国内镜像，使用cnpm替代npm加速下载，安装速度。

3.具体命令行为：

//全局安装 vue-cli
$ cnpm install –、global vue-cli
// 创建一个基于 webpack 模板的新项目
$ vue init webpack my-project
// 安装依赖，走你
$ cd my-project
$ cnpm install
$ cnpm run dev

安装过程中可能出现问题包括:node或者npm版本不够，或者端口8080被占用等情况，可自行网搜解决。
